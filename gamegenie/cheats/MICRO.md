# Micro Editor Cheat Sheet

| **Action** | **Shortcut** |
|-------------|--------------|
| Quit | `Ctrl + Q` |
| Save | `Ctrl + S` |
| Cut Line | `Ctrl + K` |
| Copy | `Ctrl + C` |
| Paste | `Ctrl + V` |
| Undo / Redo | `Ctrl + Z` / `Ctrl + Y` |
| Search | `Ctrl + F` |
| Go to Line | `Ctrl + G` |
| Select All | `Ctrl + A` |
| Toggle Line Numbers | `Alt + n` |

## Notes

- Use your **mouse wheel** to scroll.
- **Right-click** opens a small command menu.
- When exiting, if prompted to save:
  - Press **`y`** then **Enter** to save changes.
  - Press **`n`** to discard changes.
